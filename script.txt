git add
-Adds files to next commit only
git status
-Lists all files
git commit
-Then git commit is used to create a snapshot of the changes made
git remote
-The git remote command lets you create, view, and delete connections to other repositories
git push
-The git push command is used to upload local repository content to a remote repository
git branch
-Git branch is effectively a pointer to a snapshot of your changes
git checkout
-Git checkout allows you to navigate through your created branches
git merge
-Git merge puts a forked history back together again
git log
-Git log keeps track of your project and can filter information
